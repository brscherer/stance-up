# Stance Up

A local-first Muay Thai stance coach that will use the browser camera to analyze stance fundamentals and provide corrective cues.

## Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm test
npm run typecheck
npm run build
npm run lint
```

## Privacy note

The MVP is designed to process camera frames locally in the browser. No backend or video upload is planned for the first version.
